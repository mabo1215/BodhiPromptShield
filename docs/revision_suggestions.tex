\begin{itemize}

\item Deployment:

\begin{verbatim}
Ollama runtime
https://ollama.ai
\end{verbatim}

\item Example Execution:

\begin{verbatim}
ollama run llama3:8b
\end{verbatim}

\end{itemize}

\paragraph{Runtime Metadata Logging}

Each inference call records:

\begin{itemize}

\item Model version
\item Token count
\item Prompt hash
\item Hardware type
\item Runtime latency
\item Memory usage
\end{itemize}

\paragraph{Hardware Template}

\begin{verbatim}
GPU: NVIDIA A100 40GB
CPU: Intel Xeon Gold
RAM: 128GB
Runtime: Ollama v0.1.x
\end{verbatim}

This configuration enables deterministic
zero-shot semantic normalization experiments.
