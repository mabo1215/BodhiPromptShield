\subsection*{Mandatory Revision 9: Tighten the Conclusion}
Please replace the current conclusion with the following more compact version:

\begin{verbatim}
\section{Conclusion}

This paper reformulates privacy-preserving text processing as a real-time prompt privacy problem for LLM/VLM-based agents. The proposed framework protects privacy at the interaction interface before downstream inference by combining privacy-sensitive span extraction, semantic-preserving prompt sanitization, policy-aware routing, and controlled restoration.

Across the controlled CPPB protocol, the results indicate that prompt-level mediation can materially reduce direct exposure of sensitive content while preserving downstream utility better than naive redaction and generic de-identification baselines. The framework is model-agnostic, middleware-compatible, and extensible to multimodal and tool-using agent settings.

At the same time, this work should be read as a deployment-oriented systems contribution rather than as a universal privacy guarantee. Its main value lies in a practical mediation pattern with measurable privacy--utility behavior under explicit threat assumptions. Future work will focus on broader benchmark design, stronger multimodal grounding for dense and noisy document inputs, adaptive policy learning for context-dependent sensitivity, and deeper real-world evaluation with formalized attack suites and deployment audits.
\end{verbatim}
