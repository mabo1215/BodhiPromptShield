\subsection*{Mandatory Revision 7: Tighten the Conclusion}
Please replace Section VIII with the following slightly sharper version:

\begin{verbatim}
\section{Conclusion}

This paper reframes privacy-preserving text processing as a real-time prompt mediation problem for LLM/VLM-based agents. The proposed framework protects privacy at the interaction interface before downstream inference by combining privacy-sensitive span extraction, semantic-preserving prompt sanitization, policy-aware routing, and controlled restoration.

Across the controlled CPPB protocol, the results indicate that prompt-level mediation materially reduces direct exposure of sensitive content while preserving downstream utility better than naive redaction and generic de-identification baselines. The framework is model-agnostic, middleware-compatible, and extensible to multimodal and tool-using agent settings.

The main contribution is therefore a deployable mediation architecture with explicit and measurable privacy--utility trade-offs under stated threat assumptions. Future work will focus on broader benchmark coverage, stronger multimodal grounding for dense and noisy document inputs, adaptive policy learning for context-dependent sensitivity, and deeper evaluation under adversarial testing and production-scale agent execution.
\end{verbatim}
