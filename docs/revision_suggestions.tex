\subsection*{Mandatory Revision 1: Fix the Broken Formatting of Algorithm 3}
Algorithm 3 in Appendix A is still visibly broken in the compiled PDF and currently looks like token-by-token wrapped text rather than a clean algorithm block.

Please reformat Algorithm 3 completely. If needed, rewrite it using the same algorithm package and style as Algorithms 1 and 2.

Suggested replacement:

\begin{verbatim}
\begin{algorithm}[t]
\caption{Authorized Restoration of Protected Entities}
\begin{algorithmic}[1]
\Require model response $y$, secure map $K$, authorization context $\alpha$
\Ensure optionally restored response $y'$
\Procedure{RestoreEntities}{$y, K, \alpha$}
    \If{not \Call{Authorize}{$\alpha$}}
        \State \Return $y$
    \EndIf
    \State $y' \gets y$
    \ForAll{token $t$ in $y'$ such that $t \in \mathrm{Keys}(K)$}
        \State $y' \gets y'[t \mapsto K[t]]$
    \EndFor
    \State \Return $y'$
\EndProcedure
\end{algorithmic}
\end{algorithm}
\end{verbatim}
